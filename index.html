Bienvenidos a mi pagina web

En esta pagina describiremos todo lo que aprenda de las paginas web HTML y CSS principalmente.

Elementos H

Son titulos que se pueden poner de manera predeterminada, además se le agrega un número consiguiente para degradar su tamaño.

Por ejemplo: h1, h2, h3, h4, h5, h6.

Los elementos p

Se usan para escribir parrafos largos y listo calisto.

Comentarios

Los comentarios se pueden agregar encerrando lo que quieres en una sentencia algo así:

Menor que, signo de exclamación hacia abajo, guión, guión, cosa que quieres comentar, guión, guión, mayor que.

Imagenes

Estas se agregan con la etiqueta img src="(URL o Dirección de la imagen)"

Sin olvidar el atributo alt escencial para las imagenes

Links

Los links se pueden hacer de varias maneras, la más común es la de tener un texto que sea un link.

Por ejemplo podemos hacer que la palabra Monogatari sea una que lleve un link.

La etique tiene es llama "a" por anchor, que significa ancla en ingles

Lo que debemos hacer es encerrarla con una etiqueta "a href="(Link hacia donde queremos)"".

Otro uso que podemos darle; la navegación dentro de la pagina web

Simplemente tenemos que agrear un atributo id con un valor de "nombre" a la etiqueta que queramos desplazarnos y a su vez, crear un anchor con atributo de href = #nombre para que vaya hacia ese lugar.

Casillas de verificación

A diferencia de las checkboxes estas casillas deberian de poder selecionar varias opciones

Se puede marcar por defecto alguna de las opciones ya sea checkbox o casilla

Similar al atributo required este atributo llamado checked lo usamos cuando queramos tener una opcion marcada por defecto.

Value

Se le agrega el atributo value a los botones de radio y checkbox para que den al extraer la informacipon nos den el texto que queremos extraer de la respuesta.

Divisores

Estos divisores expresados con la etiqued div nos ayudan a dividir los elementos de nuestra página web para que sea mas facil manipularlos y darles cierto estilo.

Doctype

Se debe agregar !DOCTYPE al principio de un archivo HTML

Colores

Se puede cambiar el color de un texto poniendo un atributo style, pero no es bueno agregar esto debido a que a la hora de corregir cosas se entrelazan los archivos html y css.

Lo que debemos hacer es un archivo css para que ahi tengamos todas las propiedades que queramos añadir.

Cambiare el color del texto.

Tamaños

Con font-size podemos cambiar los tamaños, por ejemplo de las letras

Texto Grande.

También se puede cambiar el tamaño de las imagenes, para poder manejarlas mejor. Con los atributos width y height.

Tipo de fuente

Este atributo lo podemos cambiar con font-family, puede recibir 2 tipos de escritura, el primero que es el principal y el segundo que se usa cuando no esta disponible el principal.

Bordes

Se agregan con border-color, border-width y border-style y acepta los atributos color, grosor y tipo respectivamente.

Otro aspecto es el de border-radius que aplica una curvatura a los bordes. Se puede ver en la imagen de TougeguchiGOD
