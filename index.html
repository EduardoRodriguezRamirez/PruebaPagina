Links

Los links se pueden hacer de varias maneras, la más común es la de tener un texto que sea un link.

Por ejemplo podemos hacer que la palabra Monogatari sea una que lleve un link.

La etique tiene es llama "a" por anchor, que significa ancla en ingles

Lo que debemos hacer es encerrarla con una etiqueta "a href="(Link hacia donde queremos)"".

Otro uso que podemos darle; la navegación dentro de la pagina web

Simplemente tenemos que agrear un atributo id con un valor de "nombre" a la etiqueta que queramos desplazarnos y a su vez, crear un anchor con atributo de href = #nombre para que vaya hacia ese lugar.

Selectores

Para selectores tenemos el id (que se utiliza para darle un valor unico a un elemento) y class (que se utiliza para aplicar un mismo diseño o configuracion a varios elementos). En este caso añadiremos los selectores por atributo que se escriben [type='radio'] que en este caso seleccionaria todos los elementos de tipo radio.

Tipos de unidades

Hay 2 tipos de unidades que son las relativas y las absolutas.

Las unidades absolutas son in y mm que se refieren a pulgadas y milimetros.

Las unidades relativas son em y rem;

em se basa en el tamaño de fuente del elemento.

Heredaciones

En CSS los estilos se heredan, si en un body se tiene un color:blue entonces todo el texto que este dentro de body será azul.

También se pueden sobreescribir las configuraciones:

Heredado->Clase->id->StyleLine->!important

Uno puede sobreescribir una clase con otra clase, tomemos de ejemplo clase1 y clase2 si clase2 fue declarada despues de clase1 entonces esta sobreescribirá a clase1 sin importar el orden en que pongamos los valores en el atributo class.

Heredado se refiere a las configuraciones que se heredaron de un elemento padre. La clase se refiere a las clases que se puede crear y agregar a los elementos. Los id son los valores unicos que le damos a los elementos. Los StyleLine son configuraciones que hacemos en la misma linea de creacion del elemento. Por último el important es un atributo que se le agrega a las configuraciones para que se sepa que ese debe ser el style sin importar los otros.

Variables CSS

Podemos tener variables en CSS estas se declaran en el selector en el que se va a usar o en este caso lo declaramos en body para que se pueda usar en todo el cuerpo. Se declaran de esta manera --variable: valor; Para usar la variable tenemos que poner en donde vendría el valor esto var(--variable). Podemos hacer un refuerzo de este valor, de tal manera que quede var(--variable, black), esto le dira que en caso de que la variable no este disponible el negro será el color determinado.
