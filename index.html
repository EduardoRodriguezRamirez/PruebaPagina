Simplemente tenemos que agrear un atributo id con un valor de "nombre" a la etiqueta que queramos desplazarnos y a su vez, crear un anchor con atributo de href = #nombre para que vaya hacia ese lugar.

Selectores

Para selectores tenemos el id (que se utiliza para darle un valor unico a un elemento) y class (que se utiliza para aplicar un mismo diseño o configuracion a varios elementos). En este caso añadiremos los selectores por atributo que se escriben [type='radio'] que en este caso seleccionaria todos los elementos de tipo radio.

Tipos de unidades

Hay 2 tipos de unidades que son las relativas y las absolutas.

Las unidades relativas son em y rem;

em se basa en el tamaño de fuente del elemento.

Heredaciones

En CSS los estilos se heredan, si en un body se tiene un color:blue entonces todo el texto que este dentro de body será azul.

También se pueden sobreescribir las configuraciones:

Heredado->Clase->id->StyleLine->!important

Uno puede sobreescribir una clase con otra clase, tomemos de ejemplo clase1 y clase2 si clase2 fue declarada despues de clase1 entonces esta sobreescribirá a clase1 sin importar el orden en que pongamos los valores en el atributo class.

Heredado se refiere a las configuraciones que se heredaron de un elemento padre. La clase se refiere a las clases que se puede crear y agregar a los elementos. Los id son los valores unicos que le damos a los elementos. Los StyleLine son configuraciones que hacemos en la misma linea de creacion del elemento. Por último el important es un atributo que se le agrega a las configuraciones para que se sepa que ese debe ser el style sin importar los otros.

Variables CSS

Podemos tener variables en CSS estas se declaran en el selector en el que se va a usar o en este caso lo declaramos en body para que se pueda usar en todo el cuerpo. Se declaran de esta manera --variable: valor; Para usar la variable tenemos que poner en donde vendría el valor esto var(--variable). Podemos hacer un refuerzo de este valor, de tal manera que quede var(--variable, black), esto le dira que en caso de que la variable no este disponible el negro será el color determinado.

Las variables, para poder usarlas globalmente se pueden poner en el selector :root que vendria siendo la raiz de todo el documento.

Si dentro del documento cambias otra vez el valor de la variable, esta tendra 2 valores, 1 lo usara en el resto del documento, el otro lo usara en toda la seccion en donde lo definiste.

Mejora de compatibilidad

Esto se puede hacer, declarando estilos antes del uso de una variable.

Por ejemplo:

color:red;

color: var(--variable)

Asi los navegadores que no puedan usar las variables podrán utilizar el último estilo declarado.

Media Query

Puedes cambiar estilos de elementos si se cumple la condicion que estableciste.

@media (max-width: 350px) { h2{ color: blue;}}

Esta query dice que cuando el ancho de la pagina sea de 350 pixeles el color de los titulos h2 cambiaran de color a azul.

Si tu cambias el tamaño de esta pagina este texto cambiara de color.

Colores

Los colores puedes agregarlos de varias maneras: Hex, rgb, rgba o simplemente los nombres predefinidos.

  • Hex se ponen de manera #000000.
  • Los rgb se ponen rgb(0, 0, 0).
  • Los rgba se ponen rgba(0, 0, 0, 1).
  • Los predefinidos solo tienes que poner el nombre, como blue, red, green.

Sombras

CSS tiene box-shadow para agregar sombras:

  • Offset-x: que delimita el rango de la sombra en horizontal.
  • Offset-y: que delimita el rango de la sombra en vertical.
  • Blur-radius: que genera un desvanecido a las sombras
  • Spread-radius: Se esparce hacia los laterales.

Hover

Esta configuracion se encarga de hacer una accion cuando el elemento seleccionado pase el cursor sobre el elemento. La configuracioón es h1: hover{color:red} esto haria que todos los titulos h1 por los que pase el cursor cambien a color rojo.

Se agrego hover en este documento, aplica mayor sombra y un borde mas ancho cuando pasa por algo cuadro de este documento.

Posición

Podemos cambiar la posicion de un elemento sin afectar la distribución de los otros elementos.

Esto quiere decir que podemos mover un elemento dentro de su area, sin afectar a otras cosas.

Esto lo aplicamos en conjunto con el hover, cada cuadro que pusimos se mueve hacia la izquierda.

El segundo valor para position que es el absolute, este valor hace que los valores de top, bottom, right y left tomen de referencia la base del documento.

El tercer valor para postion es el fixed, que es similar al absolute, la diferencia es que los demas elementos ya no toman en cuenta la posicion del elemento, es decir, para la distribucion ya no existe este elemento.

Fixed puede ser utilizado para la navbar.

Float

Utilizamos float para dividir en 2 secciones lo principal, por una parte los links y por otra la informacion.

float: left; para los links, para esto agregamos la una seccion para el aside, y una seccion diferente para la informacion. Ademas cambiamos la distribucion o el espacio que utiliza cada seccion.

Z-index

Este atributo nos ayuda a poner una jerarquización para los elementos, por ejemplo, si un elemento esta sobre otro, podemos hacer que el de atras se vaya hacia delante.

Esto lo aplicamos con los cuadros que contienen información, para que la parte superior no se pusiera por detras de los cuadros de informacion.
